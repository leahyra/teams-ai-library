# Basic Configuration

After learning about the [core concepts](./../core-concepts.md) on how to enable an application in Teams, this guide will help you configure the underlying resources correctly in Azure. There are two main ways of doing this integration.

1. Using [M365 Agents Toolkit](./agents-toolkit.md) which uses a CLI/VSCode extension to help automate that work for you.
2. [Manually Configuration](./manual-configuration.md) if you have a setup that deviates from something simple
